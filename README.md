# GraphQL Professional Notes

A structured, professional-grade set of personal notes for learning and mastering GraphQL — from fundamentals through to federation, performance, security, and production-ready patterns.

## Structure

- `00-prerequisites/` — HTTP, JSON, Node.js, and database basics
- `01-getting-started/` — GraphQL fundamentals and first server
- `02-schema-definition-language-sdl/` — SDL type system
- `03-queries-and-mutations/` — Operations and best practices
- `04-resolvers/` — Resolver mechanics and testing
- `05-data-sources-and-caching/` — DataLoader, caching, N+1
- `06-subscriptions/` — Real-time GraphQL
- `07-authentication-and-authorization/` — Auth patterns
- `08-performance-optimisation/` — Query complexity, persisted queries
- `09-security/` — Hardening a GraphQL API
- `10-tooling-and-ecosystem/` — Clients, codegen, IDEs
- `11-federation-and-microservices/` — Apollo Federation v2
- `12-real-world-projects/` — End-to-end example projects
- `13-testing/` — Unit, integration, load, and contract testing
- `14-best-practices-and-patterns/` — Schema design patterns
- `15-code-snippets/` — Reusable code examples
- `16-resources-cheatsheets/` — Cheatsheets and further reading
